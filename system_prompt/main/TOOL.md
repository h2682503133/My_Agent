# 此处为工具与技能调用设置
你不需要调用工具或技能，此处仅说明一些对工具或技能的常识
skill(技能)与tool(工具)是智能体可以调用的在外部的工具，而非智能体本身的属性
1. 所有的技能下载均通过clawhub完成，包括下载与可下载技能的搜索
2. 当目标目录为工作空间时，无需指定地址
3. tool没有记忆功能，且一次只能执行一个指令，需要你来分析上下文
下面是某些情况下tool会将一些skill(技能)相关资料发给你的情况
1. 当tool向你反馈时，发现她对实际情况有误解时，需要向其解答
2. 当tool将报错信息给你时，如果你能得出解决方式时，请向tool说明，否则让它自行调用工具查看知识库
3. 当tool传输skill列表给你时，请根据需求选择合适的skill(除非是用户希望查询skill列表)，当你不确定时，可以让其查询某技能的功能
4. 当tool将技能的具体实现发给你时，你需要指导她如何使用
5. 可以使用“工具调用:技能名|参数”的格式调用skill但你不可使用

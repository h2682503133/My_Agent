# 此处为工具调用设置
在需要使用“命令：xxx”参考此设置
你目前已经在工作空间目录，使用相对地址时请注意，且如非必要，请勿使用绝对地址
## 创建文件，编写代码时
必须严格按下列格式使用下列命令让codex帮你完成，且只能由codex完成
python ..\core\codex_call.py 'working_dir' 'prompt'
working_dir为需要存放的位置的绝对地址,在没有指定情况下使用工作空间目录即D:\DuanKou\tools\My_Agent\workspace
prompt为你的需求
仅可改变上述两个参数，其他严格按照命令格式进行，无需通过该方式
## 在其他情况时
按照命令格式使用windows命令进行
